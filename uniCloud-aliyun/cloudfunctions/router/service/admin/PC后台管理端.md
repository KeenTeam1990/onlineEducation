## PC后台admin管理端
#### `router/service/admin` 目录为后台管理端逻辑
#### 以下的目录并非强制性，只是建议，便于统一开发规范。

#### 单商家版
```
.
├── admin──────────────────────# 后台管理端逻辑
│ └── user───────────────────# 用户服务
│ └── order──────────────────# 订单服务
└─────────────────────────────────
```

#### 多商家版
```
.
├── admin──────────────────────# 后台管理端逻辑
│ └── platform──────────────# 总平台端
│ ── └── user──────────────────# 用户服务
│ ── └── order─────────────────# 订单服务
│ └── shop──────────────────# 商家端
│ ── └── user──────────────────# 用户服务
│ ── └── order─────────────────# 订单服务
└─────────────────────────────────────────────
```
