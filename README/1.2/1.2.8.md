#### 1、减少APP或小程序的审核次数，页面组件局部更新不需要重新发布小程序，且实时生效。
#### 2、方便维护，只需要修改数据库对应的组件数据即可达到修改页面的目的。
#### 3、是支持可视化装修页面的必备能力。
